\chapter{Dati da completare manualmente}\label{ax:dati-da-completare}

Prima della generazione della relazione finale in Prism e opportuno completare i seguenti dati:

\begin{itemize}
    \item Periodo del tirocinio.
    \item Numero di ore.
    \item CFU.
    \item Corso di laurea.
    \item Struttura ospitante.
    \item Nome e cognome dello studente.
    \item Matricola.
    \item Anno accademico.
    \item Eventuale sito online del progetto.
    \item Eventuale link GitHub.
    \item Screenshot dell'interfaccia principale.
    \item Screenshot di uno schema ER.
    \item Screenshot della vista di traduzione.
    \item Screenshot della vista logica.
    \item Screenshot o esempio di export.
    \item Eventuali ringraziamenti personali.
    \item Eventuali riferimenti bibliografici richiesti dal corso.
\end{itemize}
