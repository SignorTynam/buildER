\begin{abstract}

This document is a preparatory dossier for the final internship report about \builder, a web application for designing, visualising, translating and exporting Entity-Relationship diagrams and logical database schemas. It summarises the project information extracted from the repository, proposes a detailed ten-page structure for the final internship report, and provides a prompt that can be used in Prism to draft the final text.

The internship was supervised by Prof.ssa Alessandra Lumini, whose role must be presented in the final report as tutor and methodological reference. The final report should therefore connect the technical work with the educational value of the internship, highlighting objectives, implemented features, technologies, testing activities, acquired skills, project limitations and possible future developments.

\end{abstract}
