buildER

Italiano

buildER è un editor web multilingua per creare diagrammi Entity-Relationship secondo la notazione Chen, progettare modelli concettuali, lavorare con viste logiche, SQL Reverse ed esportare diagrammi dal browser.

English

buildER is a multilingual web editor for Chen-style Entity-Relationship diagrams, conceptual database models, logical views, SQL Reverse workflows, and browser-based diagram exports.

Shqip

buildER është një editor web shumëgjuhësh për diagrame Entity-Relationship në stilin Chen, modele konceptuale të bazave të të dhënave, pamje logjike, rrjedha SQL Reverse dhe eksportim diagramësh nga browser-i.

  • Visual ER diagram editor / Editor visuale ER / Editor vizual ER.
  • Project files that can be saved and reopened / Progetti salvabili e riapribili / Projekte që ruhen dhe rihapen.
  • PNG, JPEG, SVG, ERS, and SQL export support.
  • Interface languages: italiano, English, shqip.
